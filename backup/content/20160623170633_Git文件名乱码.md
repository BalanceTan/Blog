### **问题**

如果文件名是中文，会显示形如`274\232\350\256\256\346\200\273\347\273\223.png`的乱码。

### **解决**

在bash提示符下输入：

``` shell
git config --global core.quotepath false
```

**core.quotepath** 设为`false`的话，就不会对*0×80*以上的字符进行*quote*。中文显示正常。
